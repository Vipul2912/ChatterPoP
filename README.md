# ChatterPop  

A real-time chat web app with theme support, encryption, and media sharing.  

## Features  
- Real-time messaging  
- 32+ adaptive themes  
- End-to-end encryption  
- File sharing  

## Tech Stack  
- **Frontend**: React, Tailwind CSS  
- **Backend**: Node.js/Express  
- **Real-time**: Socket.io
- **Database**: MongoDB 

## Setup  
```bash
git clone https://github.com/your-repo/chatterpop.git  
cd chatterpop  
npm install  
npm run dev
